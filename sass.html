Sass

Sass (Syntactically Awesome Style Sheets) er et script språk som gjør stiling av nettside enklere og mer oversiktlig, men Sass kan ikke tolkes av HTML filer og må derfor konverters til Css. Sass i utgangspunktet hjelper webutviklere å stile sidene på effektiv måte ved å gjøre css kodene oversiktlige og kortere. Det viktigste som sass innholder er veriabler. Man kan definere en veriabel med dollar tegn.

Min sass Struktur

Min struktur er slik:
prosjekt>>oblig>>sass(_var.scss, _mixins.scss og main.scss) som konverteres til prosjekt>>oblig>>css(main.css)
