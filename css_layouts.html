Hva er css-layout:

Css-layouts er nettsidens utseende. Webutviklere prøver å designe nettsiden ved bruk av css til at nettsiden ser fin og brukervennlig til brukeren. Css layouts innebarer mange metoder. Jeg kommer til å skrive litt om noen av de viktigste metodene i denne siden.

FLexbox

Flexbox er en endimensjonal layoutmetode for å legge ut elementer i rader eller kolonner. Denne metoden lar webutviklere å tilpasse innholdet i nettsiden ved å ordne og organisere de forskjellige elementene enten i rader eller kolonner og kontrollere elementenes posisjoner. Dessuten kan webutviklere kontrollere hvert element separat. Denne metoden er ev de metodene som gjør nettsiden mer rospansiv og det er viktig til all webutviklere å forstå prinsippene i flexbox.

Css-Grid

CSS Grid Layout er et todimensjonalt layoutsystem. Denne metoden lar webutviklere å tilpasse elementene i nettsiden i rader og kolonner. gjennom å dele nettsiden til rader og kolonner, kan webutviklere kontrollere nettsiden layout flexibelt og rospensivt og de kan jobbe med hvert element separat. Denne metoden er veldig nyttig og har gjort det sammen med flexbox letter til webutviklere å designe nettsidene. Css-Grid gjør nettsiden mer rospansiv hvor nettsidens innhold kan tilpasse forskjellige inheter.

Flexbox og Css-Grid er ikke absulutte

Selv om flexbox og Css-Grid har gjort det lettere til webutviklere å desgine men problemet er at de to metodene ikke tilpasser alle nettlesere. noen nettlesere og gamle versjoner kan ikke vise nettsidens innhold på riktig måte, Men Css-Layouts har ikke bare de to metodene. det finnes mange css-layout metoder.

Float

Float er en metode som lar webutviklere kontrollere elements posisjon i forhold til foreldreelementer. Denne metoden tar verdien "left, right, top og buttom".

Posisjonering

Denne metoden lar webutviklere å posisjonere elementer, og denne metoden har tre egenskaper "relativ, fixed, absulute".
