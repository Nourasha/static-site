HTML-Skjemaer

HTML-Skjema er en metode som lar brukeren til å sette inn data. Data fra brukeren som ble satt inn sendes til en web server for å behandle. Data som kan brukeren sende er for eksempel navn, passord, e-post og mye mer. HTML-skjemaer gjør nettstedet mer responsiv og kan SEO tolke nettstedet bedre.

Noen feltetyper som finnes i HTML5:

Date type input

Lar brukeren sette in dato

Date type input

Lar brukeren sette in dato

Month type input

Lar brukeren sette in dato måneder og år men ikke dager.

Email type input

Lar brukeren sette in e-post og lar innhet netleseren tilpasse tastaturen med @ og .com

Number type input

Lar brukeren sette in nummer

Range type input

Lar brukeren sette in nummer mellom to bestamte tall

Search type input

Lager et felt med søk knapp og lar brukeren søke.

texterea tag

lager et større tekst felt og Lar brukeren skrive inn for eksempel et kommentar.
